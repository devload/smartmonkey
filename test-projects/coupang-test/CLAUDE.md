**도구**: SmartMonkey AI Mode

## 🎯 테스트 미션

### 주요 시나리오
1. **상품 검색하고 장바구니에 담기**
   - 검색창에서 상품 검색
   - 검색 결과에서 상품 선택
   - 상품 상세 페이지에서 장바구니 담기

2. **카테고리 탐색**
   - 카테고리 메뉴 열기
   - 특정 카테고리 선택 (예: 식품, 패션)
   - 카테고리 내 상품 둘러보기

3. **로켓배송 상품 찾기**
   - 홈페이지에서 로켓배송 배지가 있는 상품 찾기
   - 상품 상세 확인

## 🌐 쿠팡 웹사이트 UI 가이드

### 주요 UI 요소

#### 상단 헤더
- **쿠팡 로고** (왼쪽 상단, 보통 파란색)
  - 위치: 화면 왼쪽 상단
  - 기능: 클릭 시 홈페이지로 이동
  - 중요도: ⭐⭐⭐ (항상 홈으로 돌아갈 수 있음)

- **검색창** (상단 중앙)
  - 위치: 헤더 중앙
  - 기능: 상품 검색
  - 플레이스홀더: "찾고 싶은 상품을 검색해보세요"
  - 중요도: ⭐⭐⭐⭐⭐ (주요 기능)

- **장바구니 아이콘** (우측 상단)
  - 위치: 헤더 우측
  - 기능: 장바구니 페이지로 이동
  - 표시: 장바구니에 담긴 상품 개수 표시
  - 중요도: ⭐⭐⭐⭐

#### 메인 컨텐츠

- **카테고리 메뉴**
  - 위치: 좌측 사이드바 또는 상단 메뉴
  - 항목: "식품", "패션의류/잡화", "뷰티", "출산/유아동", "가전디지털", "스포츠/레저" 등
  - 기능: 클릭 시 해당 카테고리 페이지로 이동
  - 중요도: ⭐⭐⭐⭐

- **상품 카드**
  - 구성:
    - 상품 이미지 (상단)
    - 상품명
    - 가격 (할인가 강조)
    - "로켓배송" 배지 (있는 경우)
    - 평점 및 리뷰 수
  - 기능: 클릭 시 상품 상세 페이지로 이동
  - 중요도: ⭐⭐⭐⭐⭐

- **"로켓배송" 배지**
  - 외형: 로켓 아이콘과 함께 표시
  - 의미: 빠른 배송 가능한 상품
  - 중요도: ⭐⭐⭐

#### 상품 상세 페이지

- **상품 이미지 갤러리**
  - 위치: 페이지 상단 또는 좌측
  - 기능: 스와이프하여 다른 이미지 확인

- **"장바구니 담기" 버튼**
  - 위치: 보통 하단 고정 영역
  - 외형: 파란색 또는 강조 색상
  - 텍스트: "장바구니 담기" 또는 "장바구니"
  - 중요도: ⭐⭐⭐⭐⭐ (주요 액션)

- **"바로구매" 버튼**
  - 위치: "장바구니 담기" 버튼 옆
  - 기능: 즉시 구매 프로세스 시작
  - 중요도: ⭐⭐⭐⭐

- **수량 조절**
  - 위치: 장바구니 담기 버튼 위 또는 근처
  - 구성: - (감소), 숫자, + (증가)
  - 기능: 구매 수량 설정

- **옵션 선택**
  - 위치: 상품 정보 하단
  - 예시: 색상, 사이즈 등
  - 기능: 상품 옵션 선택

#### 하단 영역

- **푸터**
  - 내용: 회사 정보, 고객센터, 이용약관 등
  - 중요도: ⭐ (테스트에서 낮은 우선순위)

### 페이지별 특징

#### 홈페이지 (/)
- 주요 프로모션 배너 (상단)
- 추천 상품 목록
- 카테고리별 상품
- 타임딜/특가 섹션

#### 검색 결과 페이지
- 검색 키워드 표시
- 필터 옵션 (가격, 배송, 브랜드 등)
- 정렬 옵션 (추천순, 낮은 가격순, 높은 가격순 등)
- 상품 그리드 목록

#### 카테고리 페이지
- 카테고리명 표시
- 하위 카테고리 메뉴
- 상품 목록

#### 장바구니 페이지
- 담긴 상품 목록
- 수량 조절
- 개별 삭제 버튼
- "선택 상품 주문하기" 버튼
- 총 주문 금액 표시

## 🤖 AI 테스팅 가이드라인

### 미션: "상품 검색하고 장바구니에 담기"

**추천 단계:**
1. 검색창 클릭
2. (검색어 입력 - 수동 또는 다른 방식)
3. 검색 결과에서 상품 선택
4. 상품 상세 페이지에서 "장바구니 담기" 버튼 클릭
5. (옵션) 장바구니 아이콘 클릭하여 확인

### 피해야 할 요소

- **광고 영역**: "광고" 표시가 있는 상품
- **URL 바**: y < 150 영역
- **푸터 링크**: 회사 정보, 약관 등
- **팝업 닫기 버튼**: 미션과 무관한 팝업은 닫기

### 우선순위가 높은 요소

1. **검색창** - 상품 찾기의 시작점
2. **상품 카드** - 미션의 핵심 요소
3. **"장바구니 담기" 버튼** - 미션 완료의 핵심
4. **카테고리 메뉴** - 탐색 경로
5. **쿠팡 로고** - 홈으로 복귀

### 좌표 힌트

- **검색창**: 보통 y 좌표 100-200 사이, 화면 중앙
- **상단 헤더**: y < 200
- **장바구니 아이콘**: 우측 상단 (x > 화면너비의 80%)
- **하단 고정 버튼**: y > 화면높이의 85%
