# Android 앱 AI 테스팅 가이드

## 🎯 최우선 처리 사항

### 0. ⚠️ 팝업/광고/모달 스마트 처리 (상황 판단 필수!)
팝업, 광고, 모달이 나타났을 때 **미션과의 관련성을 판단**하여 처리하세요.

**판단 기준**:
1. **미션과 관련 없음** → 닫기
   - 예: 미션이 "상품 둘러보기"인데 "앱 다운로드", "다른 앱 설치" 광고
   - 예: 외부 웹페이지나 다른 앱으로 이동하는 팝업
   - 예: 이벤트/프로모션 안내 (미션이 "이벤트 참여"가 아닌 경우)

2. **미션과 관련 있음** → 탐색 또는 상호작용
   - 예: 미션이 "광고 클릭 후 상품 확인"인데 광고 팝업
   - 예: 미션이 "쿠폰 받기"인데 쿠폰 안내 팝업
   - 예: 상품 상세 정보 팝업 (미션이 "상품 둘러보기"일 때)

3. **판단이 어려움** → 히스토리 확인
   - 같은 위치/같은 팝업에서 2회 이상 막혔다면 → 닫고 다른 경로 시도
   - 처음 보는 화면이면 → 미션과의 관련성을 더 신중히 판단

**팝업 닫기 방법** (미션과 무관하다고 판단했을 때):
1. **X 버튼 (닫기)**: 우측 상단이나 좌측 상단의 X, ✕, × 아이콘
2. **뒤로가기 버튼 (←)**: 화면 좌측 상단
3. **"닫기", "취소", "나중에" 버튼**: 텍스트로 표시된 닫기 버튼
4. **팝업 외부 영역 클릭**: 어두운 배경 영역 (dimmed area)
5. **Back 키 사용**: 위 방법이 안 보이면 제안

**⚠️ 중요**:
- **무조건 닫지 마세요!** 먼저 미션과의 관련성을 판단하세요
- 팝업 내용을 읽고 미션 달성에 도움이 되는지 확인하세요
- 같은 곳에서 반복적으로 막히면 전략을 바꾸세요

**우선순위**: 권한 다이얼로그 > 미션 관련 판단 > 미션 수행

### 1. 권한 다이얼로그 (Permission Dialogs)
앱 실행 시 권한 요청 다이얼로그가 나타나면 **무조건 먼저 처리**해야 합니다.

**공통 패턴**:
- "허용" / "Allow" 버튼 - 항상 클릭
- "확인" 버튼 - 항상 클릭
- "허용 안함" / "Deny" - 클릭하지 않음

**주요 권한 다이얼로그**:
1. **알림 권한**: "알림을 보내도록 허용하시겠습니까?" → "허용" 클릭
2. **위치 권한**: "위치 정보에 액세스하도록 허용하시겠습니까?" → "허용" 클릭
3. **카메라 권한**: "카메라에 액세스하도록 허용하시겠습니까?" → "허용" 클릭
4. **저장소 권한**: "사진 및 미디어에 액세스하도록 허용하시겠습니까?" → "허용" 클릭
5. **앱 이용 권한 안내**: "확인" 버튼 클릭

**⚠️ 권한 다이얼로그 버튼 위치 (매우 중요!)**:
- 권한 안내 다이얼로그의 "확인" 버튼은 **다이얼로그 하단**에 있습니다
- **화면 해상도 1080x2408 기준**:
  - X 좌표: 약 350 (화면 중앙)
  - Y 좌표: 약 **1200~1230** (다이얼로그 하단, 파란색 버튼 중앙)
- **절대 다이얼로그 중간 영역(y=800~1000)을 클릭하지 마세요** - 그곳은 권한 설명 텍스트 영역입니다
- 파란색 "확인" 버튼은 항상 다이얼로그의 **맨 아래**에 위치합니다
- 버튼을 찾을 때는 **화면 하단 1/3 영역(y > 1100)**을 중점적으로 확인하세요

**우선순위**: 권한 다이얼로그 > 미션 수행

### 2. 초기 온보딩/튜토리얼
- "다음" / "건너뛰기" / "시작하기" 버튼을 눌러 빠르게 진행
- 튜토리얼은 건너뛰고 메인 화면으로 이동

## 📱 Android 앱 공통 UI 패턴

### 네비게이션 요소
- **뒤로가기 버튼** (←): 화면 왼쪽 상단, 이전 화면으로
- **홈 버튼** (🏠): 하단 네비게이션, 메인 화면으로
- **햄버거 메뉴** (≡): 왼쪽 상단, 메뉴 열기
- **닫기 버튼** (X): 팝업/다이얼로그 닫기

### 하단 네비게이션 바
대부분의 앱은 화면 하단에 주요 메뉴가 있습니다:
- 홈 (🏠)
- 카테고리 / 탐색
- 검색 (🔍)
- 마이페이지 / 프로필
- 장바구니 / 메뉴

### 상단 앱바
- **로고**: 홈으로 이동
- **검색 아이콘** (🔍): 검색 화면
- **알림 아이콘** (🔔): 알림 목록
- **설정 아이콘** (⚙️): 설정 화면

### 리스트 및 스크롤
- **카드형 리스트**: 각 항목 클릭 시 상세 화면
- **무한 스크롤**: 아래로 스와이프하면 더 많은 항목 로드
- **새로고침**: 위에서 아래로 스와이프

## 🛍️ 쿠팡 앱 (Coupang) UI 가이드

### 메인 화면
- **검색창** (상단): "쿠팡에서 검색하세요!" - 상품 검색
- **카테고리 아이콘들** (상단 스크롤): 식품, 패션, 뷰티, 가전 등
- **로켓배송 배지**: 빠른 배송 상품
- **상품 카드**: 이미지 + 가격 + 할인율 + 별점

### 하단 네비게이션
1. **홈** (🏠): 메인 화면
2. **카테고리**: 전체 카테고리 보기
3. **검색** (🔍): 검색 화면
4. **마이쿠팡**: 주문내역, 찜한상품 등
5. **장바구니** (🛒): 담은 상품 확인

### 상품 상세 화면
- **상품 이미지**: 스와이프하여 여러 이미지 확인
- **가격 정보**: 할인가, 정상가, 할인율
- **장바구니 담기** 버튼 (하단 고정)
- **바로구매** 버튼
- **수량 선택**: +/- 버튼
- **상품 설명**: 스크롤하여 확인
- **리뷰**: "리뷰" 탭 클릭

### 검색 화면
- **검색어 입력**: 상단 검색창
- **최근 검색어**: 이전 검색 기록
- **추천 검색어**: 인기 검색어
- **필터 버튼**: 가격, 배송, 브랜드 등 필터링

## 🎮 AI 테스팅 전략

### 우선순위
1. **권한 처리**: 권한 다이얼로그가 보이면 무조건 먼저 처리
2. **미션 관련 요소**: 미션 달성에 필요한 UI 요소 우선
3. **새로운 요소**: 아직 클릭하지 않은 요소 우선
4. **핵심 기능**: 검색, 카테고리, 상품 보기 등 주요 기능 우선

### 회피해야 할 요소
- **광고 배너**: 테스트와 무관한 광고는 클릭하지 않음
- **외부 링크**: 앱 밖으로 나가는 링크 회피
- **로그인 요구**: 로그인 없이 둘러볼 수 있으면 로그인 건너뛰기
- **결제 버튼**: 실제 결제는 하지 않음

### 탐색 패턴
1. **메인 화면 둘러보기**: 주요 카테고리, 인기 상품 확인
2. **카테고리 탐색**: 여러 카테고리 들어가보기
3. **상품 상세 보기**: 상품 클릭하여 상세 정보 확인
4. **검색 사용**: 특정 상품 검색해보기
5. **다양한 화면 탐색**: 가능한 여러 화면 방문

### 좌표 선택 시 주의사항
- **상단 상태바 영역** (y < 100): 시스템 UI, 클릭 금지
- **하단 네비게이션바 영역** (y > 화면높이 - 150): 가능하지만 신중하게
- **화면 중앙 콘텐츠**: 주로 여기서 선택
- **버튼 중앙**: 버튼의 정확한 중앙 좌표 클릭
